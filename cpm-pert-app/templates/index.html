              Import File Formats
            

            
          

          

            
CSV

            

              Header row required. CPM: ac, pr, du, name.
              PERT: ac, pr, opt, ml, pess, name.
              Multiple predecessors: concatenated letters (AB) or comma-separated.
            

            
ac,pr,du,name
A,-,3,Design
B,A,5,Build
C,AB,2,Test


            
JSON

            

              Array of task objects. CPM tasks use duration;
              PERT tasks use optimistic, most_likely, pessimistic.
              dependencies may be a string or an array.
            

            
[
  { "id": "A", "name": "Design", "duration": 3, "dependencies": "" },
  { "id": "B", "name": "Build",  "duration": 5, "dependencies": "A" },
  { "id": "C", "name": "Test",   "duration": 2, "dependencies": ["A","B"] }
]


            
Excel (.xlsx)

            

              First sheet is used. Same column layout as CSV — first row must be a header.
            

            

              
                
acprduname

              
              
                
A-3Design

                
BA5Build
